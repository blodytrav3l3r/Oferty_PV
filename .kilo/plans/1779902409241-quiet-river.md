# Plan: Responsywność frontendu + polskie komentarze

## Przegląd

Projekt WITROS Oferty PV ma ~200 angielskich komentarzy w CSS, ~80 w HTML i ~200 w JS.
Responsywność istnieje w podstawowej formie (główne breakpointy w `style.css`, `studnie.css`)
ale brakuje jej w kluczowych obszarach: nawigacja, formularze na małych ekranach, tabela zleceń, tabletki kreatora.

Plan dzieli prace na **9 faz**, każda zawiera testy.

```ts
// tests/i18n/comments.test.ts
// Sprawdza, czy w plikach CSS/HTML/JS nie ma angielskich komentarzy
// Używa wyrażeń regularnych do wykrycia typowo angielskich słów w komentarzach
// Header, Navigation, Tables, Cards, Buttons, Inputs, Search, Modal, Toast, Scrollbar, Footer, Main
// Każde pojawienie się = błąd testu
describe('Komentarze po polsku', () => {
  const ENGLISH_WORDS = ['Header', 'Navigation', 'Button', 'Input', 'Table', 'Card',
    'Search', 'Modal', 'Toast', 'Scrollbar', 'Footer', 'Layout', 'Sidebar',
    'Wrapper', 'Container', 'Section', 'Panel', 'Grid', 'Row', 'Column',
    'Filter', 'Tab', 'Responsive', 'Breakpoint', 'Hover', 'Focus', 'Active',
    'Dropdown', 'Tooltip', 'Loader', 'Spinner', 'Shimmer', 'Badge', 'Chip',
    'Summary', 'Group', 'Item', 'List', 'Form', 'Field', 'Label', 'Placeholder'];

  test.each(cssFiles)('$filename nie ma angielskich komentarzy', ({ filename, content }) => {
    const comments = content.match(/\/\*[\s\S]*?\*\//g) || [];
    for (const comment of comments) {
      for (const word of ENGLISH_WORDS) {
        expect(comment).not.toMatch(new RegExp(`\\b${word}\\b`));
      }
    }
  });
});
```

## Faza 1: Responsywność nagłówka nawigacji

**Pliki:** `public/css/style.css` (linie 104-234)

**Cel:** Nawigacja z 4-5 kafelkami (`nav-tile`) nie mieści się na ekranach ≤700px.
Kafelki mają tekst (`.nav-tile-text`) i ikonę — na małych ekranach pokazujemy tylko ikony.

### Zmiany w CSS

Dodać blok `@media (max-width: 700px)` w `style.css`:

```css
@media (max-width: 700px) {
    .header-inner {
        flex-wrap: wrap;
        min-height: auto;
        padding: 0.3rem 0.8rem;
        gap: 0.3rem;
    }
    .header-left {
        flex-wrap: wrap;
        gap: 0.3rem;
        width: 100%;
    }
    .nav-apps {
        padding-left: 0;
        margin-left: 0;
        border-left: none;
        flex-wrap: wrap;
    }
    .nav-tile {
        padding: 0.3rem 0.5rem;
    }
    .nav-tile-text {
        display: none;
    }
    .header-center {
        display: none;
    }
    .header-right {
        font-size: 0.7rem;
    }
    .logo {
        font-size: 1rem;
    }
    .logo span {
        display: none;
    }
}
```

Dodać blok `@media (max-width: 480px)` dla jeszcze węższych ekranów:

```css

    .header-left {
        justify-content: center;
    }

    }
}
```

## Faza 2: Responsywność gridów formularzy

**Pliki:** `public/css/style.css` (linie 557-572)

**Cel:** `.form-row-2/3/4` schodzą do `1fr 1fr` przy 768px, ale na ≤480px powinny być `1fr`.

### Zmiany w CSS

Dodać do istniejącego bloku `@media (max-width: 768px)` w `style.css`:

```css
@media (max-width: 480px) {
    .form-row-2,
    .form-row-3,
    .form-row-4,
    .form-row-compact {
        grid-template-columns: 1fr;
    }
}
```

Dla wizardskich gridów w `studnie.css`:
```css
@media (max-width: 600px) {
    .wizard-form-grid {
        grid-template-columns: 1fr;
    }
}
```

```ts
import * as fs from 'fs';
import * as path from 'path';

export function readCssFile(relativePath: string): string {
  return fs.readFileSync(path.resolve(__dirname, '../..', relativePath), 'utf-8');
}

export function readHtmlFile(relativePath: string): string {
  return fs.readFileSync(path.resolve(__dirname, '../..', relativePath), 'utf-8');
}

export function hasMediaQuery(css: string, breakpoint: string, selector: string, property: string, value: string): boolean {
  // Uproszczona wyszukiwarka — sprawdza czy dany selektor z daną własnością
  // występuje wewnątrz media query o danym breakpoincie
  const mqRegex = new RegExp(`@media\\s*\\(max-width:\\s*${breakpoint}\\)[\\s\\S]*?\\}[\\s]*\\}`, 'g');
  const matches = css.match(mqRegex);
  if (!matches) return false;

  for (const mq of matches) {
    if (mq.includes(selector) && mq.includes(`${property}: ${value}`)) {
      return true;
    }
  }
  return false;
}
```
